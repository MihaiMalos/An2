	Se numeste graf orientat un triplet g = (N, A, g) format dintr-o multime N de elemente numite noduri sau varfuri, dintr-o familie A de elemente numite arce si dintr-o aplicatie g : A -> N x N, numita functie de incidenta prin care fiecarui element a (apartine) A i se asociaza o pereche ordonata (x, y) apartine N x N cu c != y; daca eliminam conditia x != y atunci arcul de forma (x, x) se numeste bucla, iar G se numeste graf general orientat.

Doua grafuri G1 si G2 se numesc izomorfme daca exista o bijectie (phi) de la N1 la N2 si o aplicatie (psi) de la A1 la A2

	Daca oricare pereche ordonata (x, y) (apartine) N x N este imagine a cel mult q, q>1, elemente din A, atunci G = (N, A, g).se numeste multigraf orientat (in graf exista arce paralele)
Daca in propozitia de mai sus consideram q=1, atunci obtinem definitia asa-numitului bigraf orientat (fara arce paralele).

	Se numeste un graf neorientat un triplet de forma G = (N, A, g), cu N o multime de noduri sau varfuri, A o multime de muchii, g este o aplicatie g : A -> P(2)(N) = P2(N) (reunit) P1(N). 
	Multigraf neorientat (graf simplu) are q>1, iar digraful neorientat are q=1, unde q reprezinta numarul maxim al muchiilor care au aceeasi imagine (numarul maxim de muchii paralele)
