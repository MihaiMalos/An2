Database Commands
_________________

Create database:
# create database DATABASE_NAME;

Connect to database:
# \connect DATABASE_NAME

Display all relations of a database:
# \d
It contains: view, table, sequence

Display all columns of a table:
# \d TABLE_NAME

Dispay tables of a database:
# \dt

Select all elements from a table:
# select * from TABLE_NAME;

Table Commands
______________

Modify data from table:
# alter table TABLE_NAME alter column COLUMN_NAME type DATA_TYPE;
# alter table TABLE_NAME add column CLUMN_NAME DATA_TYPE;
# update city set city = 'Brasov' where zip = 1;
